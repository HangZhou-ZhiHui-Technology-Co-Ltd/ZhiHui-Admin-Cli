## VScode

### 快捷键

| 含义         | MAC                | WIN |
| ------------ | ------------------ | --- |
| 上下移动代码 | `alt + ↑↓`         |
| 上下复制代码 | `shift + alt + ↑↓` |
| 新建窗口 | `cmd + shift + n` |
| 打开文件/文件夹 | `cmd + o` |
